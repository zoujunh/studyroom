### 📌 给 OpenCode 的明确修改指令（按钮透底效果）

> **目标**：将当前按钮（右图）改为左图效果 —— 按钮为纯轮廓+文字，**背景图完全可见、无遮罩干扰**，实现“悬浮于背景之上”的轻盈感

#### ✅ 关键原则

* **按钮自身不能有背景色或半透明遮罩**
* **禁止在按钮区域叠加任何 `::before` / `::after` 遮罩层**
* **仅保留边框 + 文字 + 微弱阴影（用于区分层次）**

#### 🔧 具体 CSS 修改要求（请替换当前按钮类名，如 `.cta-btn`）

    .cta-btn {
      /* ❌ 当前问题（右图） */
      /* background: rgba(0, 0, 0, 0.4); */
      /* backdrop-filter: blur(10px); */
    
      /* ✅ 正确写法（左图效果） */
      background: transparent;        /* 必须透明！ */
      border: 2px solid rgba(255, 255, 255, 0.3); /* 细边框，低对比 */
      color: white;
      font-size: 1.125rem;
      font-weight: 500;
      padding: 16px 32px;
      border-radius: 9999px;
      transition: all 0.3s ease;
    
      /* 轻微悬浮感（非厚重阴影） */
      box-shadow: 
        0 4px 16px rgba(0, 0, 0, 0.08),   /* 极淡投影，仅用于脱离背景 */
        inset 0 2px 4px rgba(255, 255, 255, 0.05); /* 内阴影增加立体感 */
    
      /* Hover 效果：边框变亮，不加背景 */
      &:hover {
        border-color: rgba(255, 255, 255, 0.6);
        transform: translateY(-2px);
        box-shadow: 
          0 6px 20px rgba(0, 0, 0, 0.12),
          inset 0 2px 4px rgba(255, 255, 255, 0.1);
      }
    
      /* 箭头图标样式（→） */
      &::after {
        content: "→";
        margin-left: 8px;
        font-weight: bold;
        opacity: 0.9;
      }
    }

#### ⚠️ 必须同步检查并修复的全局问题

1. **移除按钮父容器的遮罩层**  
   如果按钮外层有一个 `.hero-section::before` 或 `.overlay` 类，且设置了 `background: rgba(0,0,0,0.5)`，**请确保该遮罩层的 `z-index` 低于按钮**，或直接删除其对按钮区域的覆盖：
      /* 错误示例（导致按钮变暗） */
      .hero-section::before {
   
        content: '';
        position: absolute;
        inset: 0;
        background: rgba(0,0,0,0.5); /* ❌ 这会盖住按钮 */
        z-index: 1;
   
      }
      /* 正确做法：让按钮 z-index > 遮罩层 */
      .cta-btn {
   
        position: relative;
        z-index: 2; /* 必须高于遮罩层 */
   
      }

2. **确认背景图是直接设在 `<body>` 或 `.hero` 上，而非按钮父级**
   
   * 背景图应通过 `body { background-image: url(...); }` 或 `.hero { background: ... }` 设置
   * 按钮容器**不能**有 `background` 或 `background-blend-mode` 等干扰属性
